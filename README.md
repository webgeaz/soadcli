# soadcli
## SØAD CLI

SØAD CLI is a client for SØAD that allows you to manage your SØAD projects easily. It provides commands to pull and push transactions from the SØAD Server. By using the CLI, you get the ability to edit your transactions locally using your favorite editor and then push the changes back to the SØAD Server.

## Installation

Install:
```bash
curl -fsSL https://raw.githubusercontent.com/webgeaz/soadcli/main/install.sh | bash
```
